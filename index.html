✨ درباره سیمین

سیمین یک زبان برنامه‌نویسی ساده، زیبا و قدرتمند است که با هدف یادگیری آسان و کدنویسی سریع طراحی شده است. این زبان با الهام از پایتون ساخته شده اما دستورات آن کوتاه‌تر و ساده‌تر است تا مبتدیان بتوانند به راحتی برنامه‌نویسی را شروع کنند.

این زبان در حال تکمیل است و روزبه‌روز بهتر می‌شود!

📖 راهنمای دستورات

📌 شروع کد: همه کدها باید با .si. شروع شوند.

# شروع کد سیمین .si.

📌 متغیرها: با . شروع می‌شوند.

.name = "علی" .age = 25

📌 چاپ: با p

p "سلام دنیا!" p .name

📌 ورودی: با i

.name = i "اسمت چیه؟ " p "سلام " + .name

📌 شرط: if و el

if .age >= 18 p "بزرگسال" el p "کودک" end

📌 حلقه for:

for i = 1 to 10 p i end # با گام for i = 1 to 10 step 2 p i end # روی آرایه for item in .list p item end

📌 حلقه while:

.x = 1 while .x <= 5 p .x .x = .x + 1 end

📌 تابع: با def

def sum(a, b) return a + b end .result = sum(5, 3) p .result

📌 آرایه:

.nums = [1, 2, 3, 4, 5] p .nums[0] # ۱ p len(.nums) # ۵ .nums.append(6) # افزودن

📌 دیکشنری:

.person = {"name": "علی", "age": 25} p .person["name"] # علی p keys(.person) # ["name", "age"]

📌 تاریخ و زمان:

p time() # زمان فعلی p date() # تاریخ فعلی p year() # سال p month() # ماه p day() # روز
